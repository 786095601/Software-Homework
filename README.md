# ArmyAutomatica
Another life game program made by three stubit students.　　

简介： 军群自动机是一种无玩家参与，全程由计算机自主控制发展的格局游戏。

其中，“棋盘”的每一个格子代表一个战斗单位或一个单位的空间，多个相连的非空格子组合成一个战斗集群。诸如此类。
